## 🔧 How It Works

### File Linking
```html
<!-- In index.html, before </body> -->
<script src="js/main.js"></script>
```

### Auto-Initialization
```javascript
// main.js automatically:
// 1. Waits untuk DOM ready
// 2. Initialize all functions
// 3. Setup event listeners
// 4. Loads icons
```

### Data Attributes
```html
<!-- HTML elements mark untuk JavaScript -->
<div data-service-card>...</div>
<button data-service-next>...</button>
<div data-testimonial-item>...</div>
```
