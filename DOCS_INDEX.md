- Modern design dengan Tailwind CSS
- Zero dependencies - pure HTML + CSS + JS
- Ready to deploy to GitHub Pages

## 📝 How to Customize

### 1. Edit Content
Open `index.html` dan ubah text:
- Line 47-48: Logo name
- Line 108-113: Hero title & intro
- Line 285+: Services cards
- Line 380+: Projects
- Dll (cek QUICK_START.md untuk detail)

### 2. Edit Colors (Opsional)
- Edit CSS variable di `css/style.css`
- Atau ubah class Tailwind di HTML

### 3. Add Images
- Simpan image di `assets/` folder
- Update src paths di HTML
